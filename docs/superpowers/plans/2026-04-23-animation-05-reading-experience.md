# Animation 05: Reading Experience — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Enhance the blog reading experience with a scroll progress bar, code block fade-in, image click-to-zoom, and heading anchor links that appear on hover.

**Architecture:** Four independent enhancements, each targeting the `.prose` content area. The progress bar is a fixed-position `<div>` at the top of the page that fills based on scroll position. Code blocks get a subtle fade-in via the existing scroll reveal system. Image zoom uses a lightweight CSS + JS lightbox (no library). Heading anchor links use a `#` icon that appears on hover, generated by a rehype plugin or a client-side script that runs after page load.

**Tech Stack:** CSS transitions, client-side TypeScript, Astro `<script>` tags, Shiki (already included with Astro)

**Design spec:** `docs/superpowers/specs/2026-04-23-personal-site-design.md` — Animation Layer 1, Category 05

- [ ] **Step 1: Create the progress bar script**

Create `src/scripts/reading-progress.ts` with:

```typescript
function initReadingProgress() {
  const bar = document.querySelector<HTMLElement>(".reading-progress");
  if (!bar) return;

  function updateProgress() {
    const article = document.querySelector("article");
    if (!article) return;

    const rect = article.getBoundingClientRect();
    const articleHeight = rect.height;
    const scrolled = -rect.top;
    const viewportHeight = window.innerHeight;
    const total = articleHeight - viewportHeight;

    if (total <= 0) {
      bar.style.width = "0%";
      return;
    }

    const progress = Math.min(Math.max(scrolled / total, 0), 1);
    bar.style.width = `${progress * 100}%`;
  }

  window.addEventListener("scroll", updateProgress, { passive: true });
  updateProgress();
}

document.addEventListener("astro:page-load", initReadingProgress);
```

This calculates progress based on how far through the `<article>` element the user has scrolled.

- [ ] **Step 1: Create the heading anchors script**

Create `src/scripts/heading-anchors.ts` with:

```typescript
function initHeadingAnchors() {
  const headings = document.querySelectorAll<HTMLElement>(".prose h2, .prose h3");

  headings.forEach((heading) => {
    if (!heading.id) {
      const slug = heading.textContent
        ?.toLowerCase()
        .replace(/[^\w]+/g, "-")
        .replace(/(^-|-$)/g, "");
      if (slug) heading.id = slug;
    }

    if (!heading.id) return;

    const link = document.createElement("a");
    link.href = `#${heading.id}`;
    link.className = "heading-anchor";
    link.setAttribute("aria-label", `Link to ${heading.textContent}`);
    link.textContent = "#";

    heading.style.position = "relative";
    heading.style.paddingLeft = "1.5em";

    heading.insertBefore(link, heading.firstChild);
  });
}

document.addEventListener("astro:page-load", initHeadingAnchors);
```

This generates slugs from heading text, injects a `#` link before the heading text, and positions it to the left of the heading.

- [ ] **Step 1: Create the image zoom script**

Create `src/scripts/image-zoom.ts` with:

```typescript
function initImageZoom() {
  const images = document.querySelectorAll<HTMLImageElement>(".prose img");

  images.forEach((img) => {
    img.style.cursor = "zoom-in";
    img.addEventListener("click", () => {
      const overlay = document.createElement("div");
      overlay.className = "image-zoom-overlay";

      const zoomed = document.createElement("img");
      zoomed.src = img.src;
      zoomed.alt = img.alt || "";
      zoomed.className = "image-zoom-expanded";

      overlay.appendChild(zoomed);
      document.body.appendChild(overlay);

      requestAnimationFrame(() => overlay.classList.add("is-active"));

      overlay.addEventListener("click", () => {
        overlay.classList.remove("is-active");
        overlay.addEventListener("transitionend", () => overlay.remove());
      });
    });
  });
}

document.addEventListener("astro:page-load", initImageZoom);
```

This creates a full-screen overlay on click with the image expanded. Clicking the overlay dismisses it.
